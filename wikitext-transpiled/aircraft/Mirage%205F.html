Description[edit | edit source]

The Mirage 5F is a rank VII French jet fighter with a battle rating of 10.3 (AB/RB/SB). It was introduced in Update "Winged Lions".

Offensive armament[edit | edit source]

Main article: DEFA 552A (30 mm)

The Mirage 5F is armed with:

  • 2 x 30 mm DEFA 552A cannons, belly-mounted (140 rpg = 280 total)

Suspended armament[edit | edit source]

Main articles: SAMP Mk 81 (130 kg), SAMP Type 25 (250 kg), SAMP Type 21 (400 kg), FZ49, SNEB type 23
Main articles: AS-30 Nord, AIM-9B Sidewinder, Matra R550 Magic 1

The Mirage 5F can be outfitted with the following ordnance presets:

  • Without load
  • 2 x Matra R550 Magic 1 missiles
  • 2 x AIM-9B Sidewinder missiles
  • 1 x AS-30 Nord missile
  • 2 x 130 kg SAMP Mk 81 bombs (260 kg total)
  • 12 x 250 kg SAMP Type 25 bombs (3,000 kg total)
  • 8 x 400 kg SAMP Type 21 bombs (3,200 kg total)
  • 36 x SNEB type 23 rockets
  • 72 x SNEB type 23 rockets
  • 14 x FZ49 rockets

Pros and cons[edit | edit source]

Pros:

  • Good missiles; IR-guided Magics are very agile and have high G limits
  • Hard hitting guns
  • Decent loadout options
  • Nimble attacker

Cons:

  • Only two missiles
  • Doesn't have access to laser-guided ordnance
  • No flares or chaff of any kind
  • Frequent uptiers against more capable jets
